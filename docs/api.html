üîê Authentication

All API requests require authentication using Clerk. Include the Bearer token in the Authorization header.

POST /api/auth/sync
Sync user data from Clerk to the database
{
  "userId": "user_123",
  "email": "user@example.com",
  "firstName": "John",
  "lastName": "Doe"
}
GET /api/auth/user
Get current authenticated user information
{
  "id": "uuid",
  "clerkId": "user_123",
  "email": "user@example.com",
  "credits": 100,
  "createdAt": "2025-01-01T00:00:00Z"
}

üìä Workbooks

GET /api/workbooks
List all workbooks for the authenticated user
[
  {
    "id": "uuid",
    "name": "My Workbook",
    "sheets": [...],
    "createdAt": "2025-01-01T00:00:00Z",
    "updatedAt": "2025-01-01T00:00:00Z"
  }
]
GET /api/workbooks/:id
Get a specific workbook by ID
POST /api/workbooks
Create a new workbook
{
  "name": "New Workbook",
  "sheets": [
    {
      "name": "Sheet1",
      "rows": 100,
      "columns": 26,
      "cells": {}
    }
  ]
}
PUT /api/workbooks/:id
Update an existing workbook
DELETE /api/workbooks/:id
Delete a workbook
POST /api/workbooks/:id/export
Export workbook to Excel file (.xlsx)
Returns: Binary Excel file with full styling support
POST /api/workbooks/import
Import Excel file (.xlsx) to create new workbook
Content-Type: multipart/form-data

ü§ñ AI Operations

POST /api/ai/chat
Send a natural language command to modify workbook
{
  "message": "Create a sales report with columns for product, quantity, and revenue",
  "workbookId": "uuid",
  "conversationId": "uuid" // optional
}
Response:
{
  "reply": "I've created a sales report with the requested columns.",
  "actions": [
    {
      "type": "update_cells",
      "sheetId": "uuid",
      "updates": {...}
    }
  ],
  "conversationId": "uuid"
}
POST /api/ai/analyze
Analyze workbook data and get insights
{
  "workbookId": "uuid",
  "query": "What are the top 5 products by revenue?"
}
